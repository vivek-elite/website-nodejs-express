GIT COMMANDS TO SET THE PROJECT
Git Repository: https://github.com/danielkhan/building-website-nodejs-express

git clone --bare https://github.com/danielkhan/building-website-nodejs-express.git .git

git config --bool core.bare false

git reset --hard

git branch


SETTING THE DEV ENVIRON
npm install express

npm install -D eslint
npx eslint --init

npm install -D prettier eslint-config-prettier eslint-plugin-prettier

npm install -D nodemon

npm install cookie-session
